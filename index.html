YARinator (Yet Another RANCID)

Previous Next

Just like RANCID for Cisco, we developed a system that gets the configurations from our Network Equipments using an specific protocol obtained by snmp (ssh or telnet) and 'git commit' them, so every change in the configuration of all our equipments is being saved daily in a GitLab project. Our IP/NOC team can search for an specifid equipment and download or copy the configuration that was running at a specific day. This helped a lot with troubleshooting and set up faster an equipment after an outage.

  • Name: YARinator
  • Collaborators: Ruben Gomez, Luis Valdez, Daisy Alba, Jorge Velazquez (NetOps Engineer).
  • Technologies: KVM, Ubuntu Server 14.04, REST API, MariaDB, Ruby On Rails, Bootstrap/AdminLTE, Nginx, Puma, GitLab.
  • Languages: Ruby, JS, HTML, CSS
